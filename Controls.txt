Controls:
Main Map Controls:
Arrow Keys- Movement/Interaction
Q - Quest Book
I - Inventory Menu
E - Equipment Menu
C - Cast Magic (Combat Mode only)
Backspace - Cancel Prayer if input is empty
O - Options Menu (Untested)
Space - Wait
Z - Stats Screen
A + Arrow Keys - Attack (Combat Mode only)
1 - Save Game
2 - Load Game

Inventory Menu Controls:
Arrow Keys - Navigate Menu
E - Equipment Menu
Esc - Return to Map

Equipment Menu Controls:
Arrow Keys - Navigate Menu
I - Inventory Menu
Enter - Select Equipment for that slot
Backspace - Back to slot selection
U - Unequip from that slot
Esc - Return to Map
